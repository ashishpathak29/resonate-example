Package: com.crackerbarrel.components

Wrapper class around the HTML video tag for displaying MP4 video

Constructor
Video(id:String, left:Int, top:Int, uiStyle:VideoStyle=null, file:String, autoPlay:Boolean=false):Video

Parameters

NameTypeDescription
idString The id of the video
leftInt left position of the video
topInt top position of the video
uiStyleVideoStyle=null style data to pass to the video
fileString The video file (.mp4) to play
autoPlayBoolean=false Should the video begin playing as soon as it's loaded?

Properties
_videoObject : VideoTag
The internal reference to the html5 video tag
autoPlay : Boolean=false
Is the video set to auto play?
file : String
The video file currently loaded

Methods

pause():Void

Changes the state of the video to paused

Events

Fires when the video file has completed
onError:EventEx{src:String}
Fires when an error occurs

Fires at a regular interval as the video progresses
