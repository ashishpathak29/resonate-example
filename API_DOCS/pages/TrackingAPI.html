Package: com.crackerbarrel.tracking
Name: TrackingAPI
Extends: com.crackerbarrel.events.EventDispatcher

This class provides all functionality required to track a course and manages the tracking method. This class should not be instanced directly and is available globablly as TrackingManager

Constructor
TrackingAPI(course:Course, minScore:Int=0, maxScore:Int=100, passingScore:Int=0.8):TrackingAPI

Parameters

NameTypeDescription

minScoreInt=0 The minimum score the user can receive.
maxScoreInt=100 The maximum score the user can receive.
passingScoreInt=0.8 The score required to pass the course
