Package: com.crackerbarrel.structure
Name: Page
Extends: com.crackerbarrel.events.EventDispatcher

An object to represent a single page

Properties
absoluteIndex : Int
The index of this page in the masterPageList of the course
data : Array
The data passed to this page from the manifest
id : String
The id of the page
index : Int
The index of this page within it's parent container
languages : KeyCollection
A collection of language objects available to this page
status : Int
The viewed staus of the page as an integer: 0 not viewed, 1, started, 2 viewed

Methods

Parameter as added by the manifest. This method should not be called directly.

NameTypeDescription
